# goit-algo-hw-02

Домашнє завдання до теми **“Основні структури даних”**.

У цьому репозиторії реалізовано практичні завдання з використанням:

- **Queue** (черга) — імітація обробки заявок
- **deque** (двостороння черга) — перевірка паліндрома
- **stack** (стек) — перевірка симетричності дужок (необов’язково)

---

## Структура репозиторію

```text
goit-algo-hw-02/
├── task_1.py   # Імітація системи обробки заявок (Queue)
├── task_2.py   # Перевірка рядка на паліндром (deque)
├── task_3.py   # Перевірка симетричності дужок (stack, необов’язково)
└── README.md

```

Завдання 1 — Черга (Queue)

Програма імітує роботу сервісного центру:

автоматично генерує заявки

додає їх до черги

послідовно обробляє заявки з черги

Використано клас Queue з модуля queue.

Запуск

Windows (рекомендовано):

py .\task_1.py

macOS / Linux / або якщо працює python у PATH:

python task_1.py

Зупинка програми: Ctrl + C (або Kill Terminal у VS Code).

Завдання 2 — Паліндром (deque)

Функція перевіряє, чи є заданий рядок паліндромом:

ігнорує регістр

ігнорує пробіли

працює для рядків з парною та непарною кількістю символів

Використано deque з модуля collections.

Запуск

Windows (рекомендовано):

py .\task_2.py

macOS / Linux / або якщо працює python у PATH:

python task_2.py

Завдання 3 — Симетричність дужок (stack, необов’язково)

Програма перевіряє коректність розміщення дужок:

круглих ()

квадратних []

фігурних {}

Використано стек для зберігання відкритих дужок.

Запуск

Windows (рекомендовано):

py .\task_3.py

macOS / Linux / або якщо працює python у PATH:

python task_3.py
